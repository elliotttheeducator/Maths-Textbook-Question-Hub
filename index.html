AI settings

Which AI answers the tutor chat. By default the app talks to a hosted proxy that already has a key configured server-side, so students need nothing at all — the key below is only used as a fallback if that proxy can't be reached.

Get a free Gemini key at aistudio.google.com, an OpenAI key at platform.openai.com, or a Claude key at console.anthropic.com. Stored only in this browser, never sent anywhere except that provider's API. To let students connect without pasting anything, share a link with &provider=openai&key=YourKey added to the end (provider is optional, defaults to Gemini) — it's saved into their browser automatically on first visit.

Leave blank to use a sensible default for the selected provider. Each provider's free tier changes its daily caps over time — check their usage dashboard if you hit rate-limit errors during a lesson. Per-message cost on paid tiers is a fraction of a cent, so a whole class's daily use should still only be cents, not dollars.
